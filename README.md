# Greenfield Lab

It's a monitoring & tracking tool for BNB Greenfield Storage Providers.

## Technologies

- React + TypeScript
- Vite
- Redux Toolkit
- Ant Design
- Axios
- SCSS (7-1 pattern)
- ESLint

## Installation

```
git clone https://github.com/PolozovMichael/TIMA_3.git
cd TIMA_3
npm install
npm run dev
```
